# PALATIUM — L'ARBITRE

**Script de validation logistique inter-frégates.**
L'Arbitre ne déplace, ne copie, ne supprime JAMAIS de fichier. Il lit et valide uniquement.

> *Loi II — Étanchéité : Aucune frégate ne peut accéder aux données internes d'une autre. L'Arbitre vérifie les sas d'entrée et de sortie.*

---

## Installation

**Aucune dépendance externe.** Python 3.8+ standard library uniquement.

Modules utilisés : `os`, `sys`, `json`, `hashlib`, `datetime`, `argparse`, `pathlib`

---

## Utilisation

### Check-Out — Vérifier les fichiers OUT/ avant transfert

```bash
python PAL_ARBITRE.py --frigate F01 --mode check-out
```

Valide que tous les fichiers attendus dans `OUT/` de la frégate sont présents, non vides et conformes (JSON valide si applicable). Affiche les instructions de transfert vers les frégates destinataires.

### Check-In — Valider les fichiers IN/ après transfert

```bash
python PAL_ARBITRE.py --frigate F02 --mode check-in
```

Valide que tous les fichiers attendus dans `IN/` ont bien été reçus.
- Tout conforme → **FRÉGATE PRÊTE AU LANCEMENT**
- Fichier manquant → **FRÉGATE BLOQUÉE**

### Validate — Check-Out + Check-In combinés

```bash
python PAL_ARBITRE.py --frigate F03 --mode validate
```

### Options supplémentaires

```bash
# Spécifier un répertoire racine Drive custom
python PAL_ARBITRE.py --frigate F01 --mode check-out \
  --drive-root /content/drive/MyDrive/DRIVE_PALATIUM

# Mode verbeux — affiche tailles et hash SHA256 tronqués
python PAL_ARBITRE.py --frigate F01 --mode check-out --verbose
```

## Manifeste des Fichiers par Frégate

| Frégate | Nom | Fichiers IN attendus | Fichiers OUT attendus |
|---------|-----|---------------------|-----------------------|
| F01 | GENITOR | `maison.glb` (via SHARED) | `project_scene_config.json`, `tags_draft.json` |
| F02 | OCULUS | `project_scene_config.json`, `tags_draft.json`, `maison.glb`, `HDRI/*.hdr` | `creative_config.json`, `tags_config.json` |
| F03 | SCRIPTORIUM | `project_scene_config.json`, `creative_config.json`, `tags_config.json`, `maison.glb`, `HDRI/*.hdr` | `OUT_FRAMES/frame_*.png` (min. 1 frame) |
| F04 | EDICTA | `OUT_FRAMES/frame_*.png`, `project_scene_config.json` | `shorts_vertical.mp4` et/ou `youtube_horizontal.mp4` |

## Codes de Sortie

| Code | Signification |
|------|--------------|
| `0` | Toutes les vérifications passées ✅ |
| `1` | Fichiers manquants ou corrompus ❌ |
| `2` | Arguments invalides / erreur de configuration |
